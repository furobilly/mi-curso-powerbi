📊 Día 2: Contar y Sumar como un Pro

⏱️ 45 min

COUNT, SUM, AVG - Las funciones que más usarás

📥 Material del día:

📝 Ejercicios SQL
Descargar
🎥 Video COUNT y SUM
Ver Tutorial
Funciones de agregación:
-- COUNT: Contar registros
SELECT COUNT(*) AS TotalVentas FROM VentasAutos;

-- SUM: Sumar valores
SELECT SUM(Precio) AS IngresoTotal FROM VentasAutos;

-- AVG: Promedio
SELECT AVG(Precio) AS PrecioPromedio FROM VentasAutos;

-- MIN y MAX
SELECT 
    MIN(Precio) AS PrecioMinimo,
    MAX(Precio) AS PrecioMaximo
FROM VentasAutos;

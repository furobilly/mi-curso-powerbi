💼 Ejercicio 1: Mi Primera Conexión

⏱️ 45 min +100 XP NUEVO

Conecta Excel a Power BI y crea tu primer reporte

📋 Escenario

Rol: Analista de una tienda local de electrónicos

Objetivo: Crear un reporte básico de ventas mensuales

Datos: Archivo Excel con ventas 2023-2024

📊

Dataset: Ventas Tienda

Descargar Excel
📹

Video Tutorial

Ver Solución

📝 Tareas a Completar

  1. Conectar datos:
    • Abrir Power BI Desktop
    • Obtener datos → Excel
    • Seleccionar archivo ventas_tienda.xlsx
    • Cargar tabla "Ventas2024"
  2. Explorar datos:
    • Vista de tabla → revisar columnas
    • Identificar tipos de datos
    • Verificar calidad (nulos, errores)
  3. Crear visualizaciones:
    • Gráfico de columnas: Ventas por mes
    • Tarjeta: Total de ventas
    • Gráfico circular: Ventas por categoría
// Código DAX para tu primera medida
Total Ventas = SUM(Ventas2024[Monto])

// Promedio de ventas
Promedio Ventas = AVERAGE(Ventas2024[Monto])

// Conteo de transacciones
Número Ventas = COUNT(Ventas2024[ID_Venta])
